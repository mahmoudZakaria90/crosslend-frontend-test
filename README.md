# Crosslend-frontend-test

`Note: if end-date picker is not defined it will reflect to today's date.`

## Project setup
```
yarn
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Testing
```
yarn test:unit
```

### Lints and fixes files
```
yarn lint
```
